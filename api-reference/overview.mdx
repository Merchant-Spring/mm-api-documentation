# API Reference

The Marketplace Manager API follows REST conventions. All endpoints accept and return JSON.

## Base URL

```
https://api.marketplacemanager.com/v1
```

## Request format

- All request bodies must be JSON with `Content-Type: application/json`
- All timestamps are ISO 8601 in UTC (e.g. `2024-01-15T09:30:00Z`)
- IDs are strings
- All requests must include your API key in the `x-api-key` header — see [Authentication](/authentication)

## Response format

Successful responses return the requested resource directly. Error responses use a consistent envelope:

```json
{
  "error": "Missing or empty path parameter: organisationId"
}
```

## Pagination

List endpoints support cursor-based pagination via query parameters:

| Parameter  | Type    | Default | Description                      |
|------------|---------|---------|----------------------------------|
| `page`     | integer | `1`     | Page number                      |
| `per_page` | integer | `20`    | Results per page (max `100`)     |

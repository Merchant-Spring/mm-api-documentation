# Quickstart

This guide walks you through making your first call to the Marketplace Manager API.

## Prerequisites

- A Marketplace Manager account ([sign up here](https://app.marketplacemanager.com/register))
- An API key — see [Authentication](/authentication)
- `curl` or any HTTP client

## Step 1 — Get your API key

Follow the [Authentication](/authentication) guide to generate your key.

## Step 2 — Make your first request

List the stores connected to your organisation:

<CodeGroup>

```bash cURL
curl -X GET "https://api.marketplacemanager.com/organisations/YOUR_ORG_ID/stores" \
  -H "x-api-key: YOUR_API_KEY"
```

```python Python
import requests

headers = {"x-api-key": "YOUR_API_KEY"}
response = requests.get(
    "https://api.marketplacemanager.com/organisations/YOUR_ORG_ID/stores",
    headers=headers
)
print(response.json())
```

```javascript Node.js
const response = await fetch(
  "https://api.marketplacemanager.com/organisations/YOUR_ORG_ID/stores",
  { headers: { "x-api-key": "YOUR_API_KEY" } }
);
const data = await response.json();
console.log(data);
```

</CodeGroup>

## Step 3 — Review the response

A successful response looks like:

```json
{
  "organisationId": "YOUR_ORG_ID",
  "stores": [
    {
      "storeId": "store-1",
      "name": "My Amazon Store",
      "marketplace": "amazon"
    },
    {
      "storeId": "store-2",
      "name": "My eBay Store",
      "marketplace": "ebay"
    }
  ]
}
```

## How it works

<Steps>
  <Step title="Get an API key">
    API keys are provisioned by MerchantSpring — contact your MerchantSpring representative. Keys are **not** created from these docs. See [Authentication](/authentication).
  </Step>
  <Step title="Find your channels">
    Most endpoints identify a channel by `merchantId` + `channelId`. Call [`POST /channels`](/api-reference/channel/retrieve-a-list-of-channels) first to discover them.
  </Step>
  <Step title="Request data">
    Run a report, view performance, or read profitability data. Follow the [Quickstart](/quickstart) to make your first live call.
  </Step>
</Steps>

## Base URL

All API requests are made to:

```
https://mm-api.merchantspring.io
```

<Info>
  The API is **HTTPS only**. Every request must include your API key in the `x-api-key` header — see [Authentication](/authentication).
</Info>

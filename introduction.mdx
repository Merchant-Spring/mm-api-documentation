# Marketplace Manager API

Welcome to the **Marketplace Manager API** — your gateway to programmatically managing marketplace integrations, orders, inventory, and more.

## What you can do with the API

<CardGroup cols={2}>
  <Card title="Manage Orders" icon="cart-shopping">
    Retrieve, update, and fulfil orders across all your connected marketplaces in one unified API.
  </Card>
  <Card title="Sync Inventory" icon="boxes-stacked">
    Keep stock levels accurate and in sync across Amazon, eBay, Shopify, and more.
  </Card>
  <Card title="Track Performance" icon="chart-line">
    Access sales analytics, profit reports, and marketplace metrics programmatically.
  </Card>
  <Card title="Manage Listings" icon="tag">
    Create, update, and optimise product listings across every marketplace channel.
  </Card>
</CardGroup>

## Base URL

All API requests are made to:

```
https://api.marketplacemanager.com/v1
```

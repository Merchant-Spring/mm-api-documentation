## How it works

<Steps>
  <Step title="Authenticate">
    Generate an API key from your [MerchantSpring dashboard](https://app.merchantspring.io) and pass it as a Bearer token on every request.
  </Step>
  <Step title="Choose your endpoint">
    Browse the [API Reference](/api-reference/overview) to find the endpoint you need — orders, inventory, listings, analytics, and more.
  </Step>
  <Step title="Make your first call">
    Follow the [Quickstart](/quickstart) guide to make your first live API request in under 5 minutes.
  </Step>
</Steps>

## Base URL

All API requests are made to:

```
https://api.marketplacemanager.com/v1
```

<Info>
  The API uses **HTTPS only**. All requests must include a valid Bearer token. See [Authentication](/authentication) for details.
</Info>
